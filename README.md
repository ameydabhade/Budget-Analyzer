## Budget Analyzer and Tracker

This is a tool to track your expenses and analyze them, built with React and styled with Tailwind css.

## Demo (Desktop only)

-   LocalStorage
-   Tailwind css
-   chart.js
-   SweetAlert2
-   react-tiptool
-   react-flipMove
